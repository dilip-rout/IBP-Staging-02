Purpose Retrieves Ids of all the applications available for the currently logged user. HTTP Method GET URL https://app.infiniteblue.com/rest/api/getApplicationIds URL Parameters sessionId The session ID obtained from the body of the response when calling login . output Optional parameter specifying the output format, one of: xml (default) or json . Response Ids of all the applications available for the currently logged user. Example Sample Response: <?xml version="1.0" encoding="UTF-8"?> <resp status="ok"> <ids> <id>123456</id> <id>123490</id> </ids> </resp>
