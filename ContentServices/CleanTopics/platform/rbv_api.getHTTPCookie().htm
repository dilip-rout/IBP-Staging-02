Purpose Returns the value of an HTTP cookie sent to the server during a user interface-based update operation. Syntax rbv_api.getHTTPCookie(name) Parameters Parameter Description name The name of the HTTP cookie. Return value The value of the HTTP cookie or Null. Example The following example logs the login name stored as a cookie on the Platform Private Cloud login page (if the user chooses the Remember user name option when logging in): var x = rbv_api.getHTTPCookie("loginName"); rbv_api.log("debug", "loginName="+x);
