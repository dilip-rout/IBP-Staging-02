Purpose Creates a new relationship definition from an XML document. See Relationship XML Definition for a description of the RelationshipDef XML node and an example of its use. Syntax createRelationshipDef(string sessionId, string xmlString); Parameters sessionId A string containing the session ID obtained at log in. XMLString An XML string describing the field in an RelationshipDef node. Output Status text: Relationship R123456 has been created Permissions Required All metadata methods require administrative privileges, regardless of the view, edit, create and delete permissions set on the application or object. Establish the session by logging in with credentials for an administrative user. Example The following example creates a relationship definition: String infoMessage = binding.createRelationshipDef(sessionId, xmlString);
