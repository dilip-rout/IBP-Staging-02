Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose Displays a status message on the page. Syntax rbf_showInfoMessage(message, isError) Parameters message The message text isError Set as Boolean true in case of error messages, otherwise false. Note: This API is supported in portals as well.
