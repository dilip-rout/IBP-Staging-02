Tabs and their child menus provide navigation through Platform application pages. See Application tabs and menus for a conceptual overview of application tabs and menus. You can change a tab to be a child menu by adding a parent tab to it. See Changing a tab to be a child menu for details. You can promote a child menu to be a tab by removing its parent tab. See Promoting a child menu to a tab for details. You can customize the icons used for tabs and menus when using the Modern - Vertical Menus UI blueprint.
