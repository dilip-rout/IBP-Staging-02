A Quick Filter template defines the condition/criteria for the Quick Filter functionality. Creating a Quick Filter Template To create a new Quick Filter template, do the following: From the Object Definition page, under the Templates section, click on New Quick Filter Template. Define the Search Filter Template, Layout, Search Conditions, Fields to Search etc. Save the template, which can be ready to use under the Use Quick Filter Template dropdown. Using a Quick Filter Template To use the Quick Filter template, do the following: Access the Page Options by clicking the icon below search in the list view of application runtime page. Click the Design This Page option, which opens the page editor for the current page. From the Object Properties of the list view, select on Enable Quick Filter. Now, from the Use Quick Filter Template dropdown, select the defined search template. Click Save.
