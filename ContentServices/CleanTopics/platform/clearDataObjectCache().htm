Purpose Clears the cache of object data records from production servers. Infinite Blue recommends that you call this method after heavy use of the SOAP API to perform updates. Syntax clearDataObjectCache(sessionId); Parameters sessionId A string containing the session ID obtained at log in. Output None Permissions Required All metadata methods require administrative privileges, regardless of the view, edit, create and delete permissions set on the application or object. Establish the session by logging in with credentials for an administrative user. Example Output example: <resp status="ok"> <Msg>Cleared cache of object records</Msg> </resp>
