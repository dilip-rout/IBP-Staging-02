Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose Selects/deselects record with given ID in list view. Syntax rbf_setSelected(recordId, selected) Parameters recordId The numeric ID of record selected true or false Note: This API is supported in portals as well. Example var x = rbf_isSelected(recordId); rbf_setSelected(recordId, !x);
