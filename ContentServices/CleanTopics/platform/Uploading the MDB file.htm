To upload an MDB file: Do one of the following: From an application page: Select New Application from the Platform menu. From a setup page: Click New Application on the sidebar. The Create a New Application dialog opens. Click Create from External Data. The Import Application Metadata page opens. Click Convert MS Access Database and click Next. The Upload Database page opens. Upload the database file. Note: The file size limit is configurable. However. the file size limit 50MB is set for Platform Public Cloud. Browse to the MDB file and click Next. The Create Objects page opens. Note: The Platform conversion tool does not support MDB formats older than Access 2000.
