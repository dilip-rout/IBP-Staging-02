Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose Returns true if a checkbox with a given integration name is checked; false otherwise. Syntax rbf_isChecked(fieldName) Parameters fieldName The integration name of checkbox field Note: This API is supported in portals as well.
