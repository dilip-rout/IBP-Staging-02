You can add multiple tabs to a page and associate each tab with one or more sections on the page. To add a page tab and associate it with a section: Open the page editor for the page on which you want to add page tabs. Click +Add Tab. A new page tab appears to the right of all other page tabs. Open the tab properties, name the tab, and optionally reposition the tab. In the section you want to associate with the new tab, open its Properties menu and select the tab from the Tab drop-down field: The tab is now associated with that section. Note that you will only see a section in the page editor if its associated tab is selected. Use Page tab properties to move a tab, specify which device(s) it will render on, or delete it.
