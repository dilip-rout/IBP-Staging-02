Purpose Runs a document template on a specified object record and returns it as a base-64 encoded string. This method only works with native Platform objects. Syntax rbv_api.runTemplate(objName, Id, templateId) Parameters objName The object integration name. Id The ID of the record. templateId The original ID of the document template. Return value The template as a base-64 encoded String. Permissions required Edit permission for the selected object type.
