Purpose This Object Script method returns uploaded file as a base-64 encoded string. This method only works with native Platform objects. Formulas used in Object Script triggers can include other API methods and regular Platform template functionality, such as loops. Object Script methods have no effect if called outside of update triggers. Syntax rbv_api.getBinaryData(objName, Id, fieldName) Parameters Parameter Description objName Integration name of object Id ID of record fieldName integration name of file upload field to retrieve value from Return value A base-64 encoded string or null Permissions required View permission for the selected object type.
