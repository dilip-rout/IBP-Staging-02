You can customize the look of application pages by adding a custom header and/or footer. You can add any HTML, CSS, and/or JavaScript to customize the header and footer. If you want to use images or refer to a CSS, you should upload those as hosted files first. See Hosted files for details. You can use the same techniques to customize the sidebar for an application (see Custom sidebar ) and to customize the user interface, for example, by executing custom JavaScript, you can change the color of or hide certain Platform HTML elements. See Working with templates , Formulas , and Client-side JavaScript for more information. Follow these steps to modify the header and footer: Navigate to the application definition. For example, From the application switcher, select Setup Home. In the Application Setup section, click Applications. Click the name of the application you want to modify. From the More actions menu, select Header And Footer. Enter your custom style information in HTML Header and HTML Footer boxes. Click Save.
