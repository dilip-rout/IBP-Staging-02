Purpose This API retrieves PROD and STORAGE component load status runtime information of a given customer. HTTP Method GET URL https://app.infiniteblue.com/rest/api/getCustomerRuntimeInfo URL Parameters id The ID of the Customer record. sessionId The session ID obtained from the body of the response when calling login . Permissions Required Full administrative privileges. Response PROD and STORAGE runtime information for the customer in JSON format. Sample Example output: [ { "componentType": "PROD", "componentName": "PROD2", "status": "LOADED" }, { "componentType": "STORAGE", "componentName": "", "status": "NOT_LOADED" } ]
