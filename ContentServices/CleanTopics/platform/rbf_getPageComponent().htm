Purpose Charts, grids, and tabular reports are represented as PageComponent objects in the new UI. This function returns a PageComponent object for a given component. The PageComponent object can then be used to refresh an individual page component or to access its Kendo configuration object. See PageComponent for more information about PageComponent and its interface. Note: This API and the PageComponent object are only available in the New UI. Syntax rbf_getPageComponent(componentId) Parameters componentId For charts and grids, the original ID of the section that contains the chart or grid (available in the section's Properties menu). For reports, the original ID of the report (available in the System Information section when viewing the report). Note: This API is not supported in portals.
