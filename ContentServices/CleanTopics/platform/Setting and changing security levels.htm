Security settings apply per customer tenant and control login session behavior and password requirements. Built-in security levels describes the available levels. Security levels apply only when using Platform password authentication. To set and change security settings for your tenant, follow these steps: Navigate to the Setup home page: From an application page, do one of the following: Select Setup Home from the Platform menu . Select Setup Home from the application switcher From a setup page, select Setup Home from the application switcher. In the Administration Setup section, click Authentication. Verify that the Authentication Type is Password and click Next. From the Security Level drop-down, select the security level. Note: If you update the Security Level, certain users in your tenant may not be able to log in using their existing credentials. Because of this, Infinite Blue recommends that you reset passwords for all your users from the Users object tab using More actions...>Reset Password. Platform resets the passwords and sends an email to all your users about their updated password. Alternatively, you can inform your users about the change of security levels in your tenant and request them to change their passwords if necessary. Click Save. A message will confirm an update to the security level.
