After installing Platform Private Cloud: If you are using a database other than OpenEdge, you need to run the scripts for that database and enter information about it in the node-config.json file. Then, you can start the runtime components as described in Starting components and logging In . The default Web server configuration may not completely secure your Web server. For example, depending on your environment, unauthorized users could access your data on the servers. Infinite Blue recommends that you optimally secure your Web servers by conforming to the benchmarks given by Center for Internet Security (CIS). For information on CIS security benchmarks, see http://benchmarks.cisecurity.org/downloads/browse/?category=benchmarks.servers.web .
