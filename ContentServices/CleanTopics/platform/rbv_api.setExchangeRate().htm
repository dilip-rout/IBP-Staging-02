Purpose Sets an exchange rate between two currencies on a given date. Syntax rbv_api.setExchangeRate(srcCode, destCode, rate, date) Parameters srcCode Currency code to convert from destCode Currency code to convert to rate Exchange rate, a decimal value, to be used for conversion. date Date for exchange rate (Defaults to current date) Return value None Permissions required Requires full administrative permissions. Example var rate = rbv_api.setExchangeRate("EUR", "USD", 1.23, new Date());
