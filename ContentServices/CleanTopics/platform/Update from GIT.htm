The More Actions menu on the application setup page includes an Update from GIT option. Use this option to upload a Platform application XML file and install updates from any GIT repository configured by the MASTER/TENANTadmin. On selecting Update from GIT returns to the GIT screen where the user has a provision to update the application from the available list. From an application settings: Select Update from GIT from the More Actions menu. The GIT page opens. Select the application you prefer to update from the GIT Repository. The Update Application from XML page opens. Select the preferred components of the application and click Update. For more information, see Installing and updating applications from XML .
