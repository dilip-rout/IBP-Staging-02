Infinite Blue Platform's Zapier app provides a dedicated integration with Zapier which is an automation tool to connect your application and other services. Zapier acts as medium between our platform and any multiple applications or services to integrate and provide automation to any redundant tasks. You can achieve any integration with minimal coding or by being less dependable on any developer. Learn more about this integration in the following sections. Before you begin: Make sure you have the following: An account in Infinite Blue SaaS Platform . An access to at least one application with at least one object. All applications should have Integration Names. Object’s default view should have an id column.
