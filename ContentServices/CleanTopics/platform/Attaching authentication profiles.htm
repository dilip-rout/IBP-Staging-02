When exporting an application, you can choose to attach an authentication profile only if you want it to be imported into the destination tenant. To attach authentication profiles: From the application switcher drop-down available next to the application name : To navigate to settings for the current application, click App Settings. To navigate to settings for a different application, hover your mouse pointer over the application and click its associated Application Settings icon. Scroll down to the Authentication Profiles section. Click Attach Authentication Profiles. From the available list of Authentication Profiles, select the profiles you want to attach. When you finish adding profiles, click Save. When importing an application, The existing Password or OE authentication profiles in a tenant will be overridden with the incoming Password or OE authentication profiles. The default authentication profile of a role will be mapped to the default authentication profile of the target tenant. Note: Deselecting an authentication profile is not allowed while importing an application xml.
