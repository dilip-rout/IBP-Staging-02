Hosted files can be associated with one or more applications. You can modify text-based hosted files, such as JavaScript, HTML, or XML, by simply modifying the text within Platform without uploading a new file. To upload and manage hosted files from setup: Navigate to the Setup home page: From an application page, do one of the following: Select Setup Home from the Platform menu . Select Setup Home from the application switcher . In the Applications Setup section, click Hosted Files. To create a new hosted file, click New Hosted File and: Enter a display name. Browse to upload the file (the size of a single uploaded file cannot exceed maximum allowed for Platform installation (5MB by default)). Optionally enter a description. Select the applications to which this file should be attached.
