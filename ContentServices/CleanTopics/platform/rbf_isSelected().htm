Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose Returns true if record with given id is selected (check box checked) in list view. Syntax rbf_isSelected(recordId) Parameters recordId The numeric ID of record Note: This API is supported in portals as well.
