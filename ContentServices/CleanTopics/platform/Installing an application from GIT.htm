You can install a Platform application from any GIT repository if configured by the MASTER/TENANT admin. To install a Platform application from GIT: Do one of the following: From an application page: Select New Application from the Platform menu. From a setup page: Click New Application on the sidebar. Select Create from External Data. The Import Application Metadata page opens. Select GIT and click Next. The GIT page opens. Select the application you prefer to install. The Install Application from XML page opens. Select the preferred components of the application and click Install. For more information, see Installing and updating applications from XML .
