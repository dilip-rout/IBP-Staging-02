You can assign a landing page to a role. The configured landing page includes the application page and the tab to open on that page. When a user with that role logs into Platform, the configured page opens with the configured tab selected. To assign a landing page to a role: Navigate to the Setup home page: From an application page, do one of the following: Select Setup Home from the Platform menu . Select Setup Home from the application switcher . Select Roles. Select Pages next to the role to which you want to assign a landing page. The Assign Page Versions page opens. Select the Configure landing page for the role check box. From the Select Application drop-down list, select an application. From the Select Tab drop-down list, select a tab. Click Save. When a user with that role logs in, the configured application will open with the configured tab selected. A user can override the landing page configured for their role on the My Preferences .
