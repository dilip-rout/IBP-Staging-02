To use LDF permissions on object records, you must enable the Organization attribute on the object type. To enable this attribute: From the application switcher drop-down available next to the application name: To navigate to settings for the current application, click App Settings. To navigate to settings for a different application, hover your mouse pointer over the application and click its associated Application Settings icon. Click Objects in the ribbon. Click Edit next to the object to modify. In the Object Attributes table, check Organization. Click Save. You can now assign LDF permissions on individual records of that type.
