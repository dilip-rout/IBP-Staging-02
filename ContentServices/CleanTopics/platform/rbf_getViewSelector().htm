Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose This function returns the view ID used by a lookup field for both auto-complete and pop-up selectors. Syntax rbf_getViewSelector(fieldName) Parameters fieldName The integration name of lookup field Note: This API is supported in portals as well.
