The node-config.json file in the config directory of the Platform Master Server instance specifies configuration for the database(s) used by Platform. The configuration includes connection information determines which database will store data for a particular customer tenant. If you change the file manually, you must restart the Web server. An administrator logged into the master tenant can create and update system databases from the System Console > System > Databases without restarting the Web server.
