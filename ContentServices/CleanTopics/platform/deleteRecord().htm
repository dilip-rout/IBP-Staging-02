Purpose Moves an object record to the recycle bin. This method works with external object records (including those mapped to OpenEdge services) and native Platform object records. Syntax deleteRecord(string sessionId, string objDefName, string uid); Parameters sessionId A string containing the session ID obtained at log in. objDefName A string containing the object definition integration name. uid A string containing the record ID. Output None. Permissions Required Delete permission for the requested object type.
