Purpose Updates an existing field definition from an XML document. See Field XML Definition for a description of the DataFieldDef XML node and an example of its use. Syntax updateFieldDef(string sessionId, string xmlString); Parameters sessionId A string containing the session ID obtained at log in. xmlString An XML string describing the field in an DataFieldDef node. Output Status text: Field ABC has been updated Permissions Required All metadata methods require administrative privileges, regardless of the view, edit, create and delete permissions set on the application or object. Establish the session by logging in with credentials for an administrative user. Example The following example shows how to invoke updateFieldDef(): String infoMessage = binding.updateFieldDef(sessionId, xmlString);
