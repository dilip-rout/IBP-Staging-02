Purpose Terminates the specified API session. This method should be called to explicitly end each Web API session. The output parameter is optional. HTTP Method GET URL https://app.infiniteblue.com/rest/api/logout URL Parameters sessionId The session ID obtained from the body of the response when calling login . output Optional parameter specifying the output format, one of: xml (default) or json . Response Standard success or failure output Example Sample XML response: <?xml version="1.0" encoding="utf-8" ?> <resp status="ok"> </resp>
