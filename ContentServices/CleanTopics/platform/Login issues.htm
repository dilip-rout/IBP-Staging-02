If you run into a problem, fix your email settings, restart Tomcat, and use the Forgot Password link (available from the login.jsp page). The system will reset your password and send another email to you. In addition to creating the first administrative user account, the system will create a second administrative account. Use this account if you are having problems logging in as the first administrative user. Delete or disable this account when you no longer need it: User name: admin2 Password: welcome
