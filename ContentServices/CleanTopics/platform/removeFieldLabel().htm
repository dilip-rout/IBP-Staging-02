Purpose Removes the specified field label from the current View or Edit page. This is useful for cases where the field label takes up extra space and/or is not necessary, such as when displaying an image. Note: All metadata methods require administrative privileges, regardless of the view, edit, create and delete permissions set on the application or object. Establish the session to use these methods by logging in with credentials for an administrative user. Syntax rb.newui.util.removeFieldLabel(fieldName) Parameters fieldName The integration name of the field Return value None Example The following example removes the field label "photo" from the current View or Edit page. rb.newui.util.removeFieldLabel("photo");
