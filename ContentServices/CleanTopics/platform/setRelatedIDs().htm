Purpose For external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection), this method assigns an array of related records to the specified record. Note: For native Platform objects, you can use the method, setRelationship() . Syntax setRelatedIDs(string sessionId, String objDefName, String uid, String relName, StringArr arr); Parameters sessionId A string containing the session ID obtained at log in. objDefName A string containing the object definition integration name. uid A string containing the record ID. relName Relationship's integration name arr IDs of related records Output None Permissions Required Edit permission for the requested object type. Example String[] arr = new String[] { "100890", "100891" }; binding.setRelatedIDs(sessionId, "client", "123456", "R1223456", new StringArr(arr));
