Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose This function returns a list of comma-separated integration codes corresponding to the currently selected options in a picklist, multi-select picklist, or group of checkboxes field. This function works in view and edit pages. Syntax rbf_getPicklistCodes(fieldName) fieldName The integration name of field Note: This API is supported in portals as well.
