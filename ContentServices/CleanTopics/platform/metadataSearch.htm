Purpose Searches for a string in metadata and returns results in XML format. HTTP Method GET URL https://app.infiniteblue.com/rest/api/metadataSearch URL Parameters sessionId The session ID obtained from the body of the response when calling login . query A text pattern to search. Response An XML document with nodes for each metadata entity found for the search string. Permissions Required Full administrative permissions. Example Sample XML response: <resp status="ok"> <SearchResults> <SearchResult id="376768" name="AJAX API Test" type="Application" updatedAt="2014-08-22T18:57:41Z" /> <SearchResult id="376879" name="AJAX Test" type="Page" updatedAt="2014-08-25T21:16:27Z" /> <SearchResult id="376881" name="AJAX Test" type="Menu" updatedAt="2014-08-22T18:58:20Z" /> <SearchResult id="376825" name="All Tests" type="View" objDefName="test" updatedAt="2014-08-22T18:57:41Z" /> </SearchResults> </resp>
