You can enable and disable reports per object in the object definition. The Reports Enabled checkbox is available under Object Properties and is checked by default. To change the value, click Edit. If you are using any right to left (RTL) languages in your report, you can specify that content is rendered RTL. See RTL support in templates and reports Right to left support in reports for details. Note: To enable Object Type Attribute for Reports, see Extending Object with Object Types .
