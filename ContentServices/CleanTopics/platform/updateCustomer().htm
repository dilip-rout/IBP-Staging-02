Purpose Updates a Customer record with the supplied values. Syntax updateCustomer(string sessionId, long id, arr, useIds); Parameters sessionId A string containing the session ID obtained at log in. id The customer ID. arr Values for fields to be updated. useIds A boolean value: if true , use numeric IDs for lookup and picklist values; otherwise, use integration codes and record names. Output None Permissions Required Edit permission for the Customer type. Example DataField[] arr = new DataField[1]; arr[0] = getField("companyName", "API Updated Test"); binding.updateCustomer(sessionId, 123456, new DataFieldArr(arr), true);
