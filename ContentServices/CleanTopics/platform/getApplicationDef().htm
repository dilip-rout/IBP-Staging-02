Purpose Retrieves a full description of a Platform application definition as an XML document. The XML schema for this document can be found at: https://app.infiniteblue.com/webapi/wsdl/metadata.xsd See Application XML Elements for a description of the Application XML node and an example of its use. Syntax getApplicationDef(string sessionId, string appId); Parameters sessionId A string containing the session ID obtained at log in. appId The original ID of the application to retrieve. Output XML document with the application definition in an Application node. Permissions Required All metadata methods require administrative privileges, regardless of the view, edit, create and delete permissions set on the application or object. Establish the session by logging in with credentials for an administrative user. Example The following example retrieves an XML description of an application. String xmlData = binding.getApplicationDef("7ae747a0e36648ef8bd016eec1502779@46216462","7567");
