Purpose Retrieves the client IP address. This API will be available for Platform Application pages, Portal pages (authenticated/un-authenticated), and Setup pages. Syntax rbf_getClientIP() Parameters None Return Value IP address Example The following example retrieves the IP of the client as the page loads. <script> $(document).ready(function () { yourFunction(); }); function yourFunction(){ var ip=rbf_getClientIP(); console.log("-------------------------->"+ip) } </script> Note: This API is supported in portals as well.
