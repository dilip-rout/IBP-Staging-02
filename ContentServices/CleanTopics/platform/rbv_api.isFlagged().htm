Purpose Determines whether a particular user flagged a record. Tip: Use the token {!#CURR_USER.id} to retrieve the ID of the currently logged-in user. This method only works with native Platform objects. Syntax rbv_api.isFlagged(objName, Id, userId) Parameters objName Integration name of object Id ID of record userId ID of the user who might have flagged the record Return value true if a record was flagged
