Purpose Similar to create() , creates a new record. Returns the ID of the newly created record. This method works with external objects (including those mapped to OpenEdge services) and native Platform objects. Syntax createRecord(string sessionId, string objDefName, DataFieldArr arr, string useIds); Parameters sessionId A string containing the session ID obtained at log in. objDefName A string containing the object definition integration name. arr A DataFieldArr array of values for fields of a newly created record. Record ID and auto-numbers are assigned automatically. useIds A boolean value: if true , use numeric IDs for lookup and picklist values; otherwise, use integration codes and record names. Output The ID of the newly created record as a string. Permissions Required Create permission for the requested object type.
