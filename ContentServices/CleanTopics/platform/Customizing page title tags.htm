A hosted or Private Cloud ISV account gives you the option to customize the <title> tag used in application pages that will be accessed by your customer tenants. Valid templates may include any text and the following merge tokens that get replaced at runtime: {!A} — Current application name {!S} — ISV system name (see above) {!P} — Customer's subscription plan name (Solo, Professional, etc.) {!O} — Current object name (if any) {!R}—- Current record name (if any)
