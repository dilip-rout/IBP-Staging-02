Purpose Starts the approval process on a selected record with the Approval attribute. See Approvals for more information about the approval process. Syntax rbv_api.startApproval(objName, Id, actionId) Parameters objName The object integration name. Id The ID of the record. actionId The original ID of the workflow action that started the approval process (can be found on the workflow action's view page). Return value None. Permissions required The current user must have Access permission on the workflow action that starts the approval process. Example rbv_api.startApproval("item", {!id}, "123456");
