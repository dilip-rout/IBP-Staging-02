Purpose Updates an existing field definition from an XML document. See Field XML Definition for a description of the DataFieldDef XML node and an example of its use. HTTP Method PUT or POST URL https://app.infiniteblue.com/rest/api/updateFieldDef URL Parameters sessionId The session ID obtained from the body of the response when calling login . xmlString An XML description of the field in a DataFieldDef node.
