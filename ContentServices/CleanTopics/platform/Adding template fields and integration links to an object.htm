Template fields are similar to template-based HTML components, but can be used in views and page layouts because they are part of an object definition. Integration links require URL encoding. The Platform template engine automatically selects the correct encoding rule, but you must enter URLs using the correct syntax. You can use template fields in combination with other field types to deeply customize the Platform user interface (see the example below). Template fields have an option to hide the display label that is normally shown on the left in pages. This feature is especially useful for creating a single piece of JavaScript code to be added to multiple pages.
