Purpose Returns the position (0-based index) of the first argument in a variable length list of remaining arguments. Syntax rbv_api.inArgs(x, arg0, arg1, …) Parameters x variable args One or more arguments Return value true or false Example rbv_api.inArgs("Z", "X", "Y", "Z") returns 2 rbv_api.inArgs("A", "X", "Y", "Z", "1") returns -1 (i.e. not found)
