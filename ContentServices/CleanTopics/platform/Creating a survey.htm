Follow these general steps to create a survey: Create a new object with a Survey attribute or enable the Survey attribute on an existing object . If you create a new object with the Survey attribute, Platform adds a Questions option to the page menu automatically. If you enable the Survey attribute on an existing object, you can add the Questions option by editing the object tab as described in Enabling surveys on an existing object . The Questions menu option (shown below) allows you to create a questions library. Platform saves survey questions in a shared library that can be accessed from any object that has the Survey attribute. To distribute the survey to end users in your tenant, make sure the Survey Taker attribute is enabled on the User object. Create survey questions as described in Creating a question library and Adding questions to a survey . Distribute the survey link in one of the ways described in Survey pages and links . See how survey takers responded as described in Collecting survey answers .
