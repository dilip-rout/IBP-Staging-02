Purpose Moves Platform object records to the Recycle bin. Deletes external objects, including those mapped to OpenEdge Service objects. HTTP Method DELETE or GET URL https://app.infiniteblue.com/rest/api/deleteRecord URL Parameters sessionId The session ID obtained from the body of the response when calling login . objName The object integration name. id The record ID. output Optional parameter specifying the output format, one of: xml (default) or json . Permissions Required Delete permission for the requested object type. Response Standard success or failure output. Example Output example in XML format: <resp status="ok"> <Msg>Record has been deleted.</Msg> </resp>
