Needs tech review Purpose Creates an log entry in the specified system log file. Note: The log() method can be used for debugging delayed triggers, SOAP calls, and REST calls where UI-based debugging is unavailable. Syntax rbv_api.Log (logName, text) Parameters logName The name of one of the system log files—webapi, rest, jobs, debug—where your log entry will be recorded. text The text entry to be logged. Return value None Example var x = rbv_api.selectValue(...); rbv_api.log("debug", "x="+x);
