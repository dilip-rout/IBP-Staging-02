Purpose This API will return settings of an authentication profile based on the given authentication profile ID for viewing. HTTP Method GET URL https://app.infiniteblue.com/rest/api/getAuthenticationProfileById URL Parameters authProfileId The authentication profile ID of the profile which has to be edited. Permissions Required Full administrative privileges. Response The authentication profile by Id for the tenant in XML format.
