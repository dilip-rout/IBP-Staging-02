To run the installer in console mode: Open a command prompt in the directory that contains the installation executable. Run the following command: cmd > PLATFORM_FULL_INSTALLER_<version>_WIN_64.exe -i console Follow the prompts as directed. See Installing Platform using the default mode for a description of the information you need to enter.
