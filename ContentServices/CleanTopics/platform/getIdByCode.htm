Purpose Retrieves the ID of a picklist item or status by providing its integration code. Note: This API only works when the ID is numeric. Therefore, Picklist(char) or Radio Button(char) are not supported. HTTP Method GET URL https://app.infiniteblue.com/rest/api/getIdByCode URL Parameters sessionId The session ID obtained from the body of the response when calling login . objName The object integration name. field The field integration name. code Integration name of picklist item or status. output Optional parameter specifying the output format, one of: xml (default) or json . Permissions Required View permission for the requested object type. Response ID of item with matching integration code or -1 Example Output example in JSON: { "id": 4567 }
