Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose This function collapses or expands the Page's section. It will have no effect if the section does not exist or is non-collapsible. Syntax rbf_setSectionCollapse(sectionId, collapsed) Parameters sectionId Original ID of Page's section (can be found in Page Editor) collapsed If true, collapse section, if false, expand section Note: This API is supported in portals as well.
