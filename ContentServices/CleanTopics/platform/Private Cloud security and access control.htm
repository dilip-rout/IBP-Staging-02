Platform Private Cloud includes all of the security features of hosted Platform. However, with Private Cloud, you can configure the following: Authentication method — You can use Platform's password authentication or you can use an external authentication engine. Security levels and policies — Using Platform's password authentication, you can customize the security level, password expiration policy, and password validation, and you can set up security questions for authentication. API authentication — When logging into Platform using an API, you can choose the authentication mechanism to use for your selected authentication method. Protocol — You can configure Platform Private Cloud to use the HTTPS protocol. For information about general security features, see Security and access control .
