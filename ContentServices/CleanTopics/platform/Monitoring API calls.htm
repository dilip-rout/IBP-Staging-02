You can view information about API access to your tenant by navigating to Setup Home> Application Setup > API. The API page provides the following information for monitoring: Runtime Info Maximum number of API calls per hour Number of REST API and SOAP API calls The time the API counter was reset Note: The API counters are set at the time you log into Platform. For example, if you log into Platform at 9:20 am, your API counters are set at 9:20 am and then the API counters are reset every 60 minutes. A link to API Logs. This link opens the Monitoring setup page, which has links to API log files—SOAP API log, REST API log, and Progress Data Service API log. The log files record every SOAP (webapi.log), REST (rest.log), and Progress Data Service (progressdatacatalog.log) API call. You must select the Enable API Log check box from Setup > Administration Setup> Account Settings to see detailed log records for API calls.
