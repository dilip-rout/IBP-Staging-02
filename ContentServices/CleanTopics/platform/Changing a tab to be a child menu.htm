You can change a tab to be a child menu by adding a parent tab to it. It will appear in an application page as a submenu to a tab. To change a tab to a menu: Navigate to an application setup page that uses the tab you want to change: To navigate to settings for the current application, click App Settings from the application switcher. To navigate to settings for a different application, hover your mouse pointer over the application in the application switcher and click the settings icon. Click Tabs in the ribbon. From the Tabs table, click Edit in the row for the tab you want to modify. A page opens that allows you to edit tab properties. From the Parent Tab drop-down menu, select the tab on which you want this tab to appear as a menu. Click Save.
