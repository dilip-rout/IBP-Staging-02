Purpose Retrieves the integration code of a picklist item or a status by providing its ID. Note: This API only works when the ID is numeric. Therefore, Picklist(char) or Radio Button(char) are not supported. HTTP Method GET URL https://app.infiniteblue.com/rest/api/getCodeById URL Parameters sessionId The session ID obtained from the body of the response when calling login . objName The object integration name. field The field integration name. id Numeric ID of picklist item or status. output Optional parameter specifying the output format, one of: xml (default) or json . Permissions Required View permission for the requested object type. Response Integration code of item with matching ID or null Example Output example in JSON: { "code": "CREATED" }
