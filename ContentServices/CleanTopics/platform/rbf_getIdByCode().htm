Warning: Support for using this method with external objects (such as those mapped to external tables, to OpenEdge Service objects, or through a HDP connection) is a beta feature. This method is supported in production systems, except for external objects. Purpose Returns the ID of an item from a picklist, status, radio buttons, or group of checkboxes field for the specified code. Syntax rbf_getIdByCode(objName, fieldName, code, callback) Parameters objName Object definition integration name fieldName The integration name of field code The integration name of the field item callback The callback function that will receive the ID of field item as first parameter (-1 if not found). Note: This API is supported in portals as well.
