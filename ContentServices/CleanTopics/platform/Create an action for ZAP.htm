Creating an Infinite Blue Platform Action in Zapier In the Do this section, Select the Infinite Blue Platform application from Your Apps section. In the Choose App & Event section, select Create Record from Choose Action Event dropdown and click Continue. You are now prompted to Sign in to Infinite Blue Platform account under the Choose Account section. If you are already signed, select the existing account, and click Continue. In the Customize New Record section, Select Application Name and Object Name. Construct a JSON string for object creation. You can map the field values from the trigger event. Sample JSON String: Click TEST & REVIEW where a test record is created in the Infinite Blue Platform. Verify the Infinite Blue Platform for the newly added record. Once done, click RETEST & CONTINUE and finish the setup by clicking on Done Editing. Turn the ZAP ON, to start using it. You have now successfully created an action to add a new record in the Infinite Blue Platform using Zapier.
