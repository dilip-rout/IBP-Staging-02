This sample app demonstrates how to use Platform workflows to automate business processes. The self-documented app describes how to exercise the workflow functionality and how to add a workflow to your application. Pre-requisites You must have an active Platform hosted or Private Cloud account with an Administrator role. Instructions Download the workflow-sample_v72 . Log into your Infinite Blue Platform account. From an application page, such as Platform Home, open the Platform menu. Select New Application. Click Create from External Data. Select Infinite Blue Platform XML File as the type to import from and click Next. Click Choose File, browse to the location of the downloaded sample, select it and click Next. The Install page displays. Click Install. Follow the instructions on the app's introduction page to exercise the application.
