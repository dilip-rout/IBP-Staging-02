This self-documenting sample app demonstrates how to use the Kendo PivotGrid widget to enhance your Platform applications. Pivot grids allow end-users to better visualize data from multiple objects. Pre-requisites You must have an active Platform hosted or Private Cloud account with an Administrator role. Instructions Download the pivotgrid-sample_v122 . Log into your Platform account. From an application page, such as Platform Home, open the Platform menu. Select New Application. Click Create from External Data. Select Infinite Blue Platform XML File as the type to import from and click Next. Click Choose File browse to the location of the downloaded sample, select it and click Next. The Install page displays. Click Install. Follow the steps on the app's Introduction & Instructions page to try out the functionality.
